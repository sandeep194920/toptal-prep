## Toptal Test Format

- **3 problems** in 90 minutes
- **All must pass** 100% of test cases
- **Patterns covered**: Typically 2-3 from the 9 core patterns
- **Difficulty**: Similar to Codility lessons

## Study Plan Suggestion

1. **Week 1-2**: Master all 9 patterns (2-3 hours per pattern)
2. **Week 3**: Practice mixed problems, optimize solutions
3. **Week 4**: Take mock tests, review weak areas
4. **Final Week**: Speed practice and edge case review

## Mock Test Strategy

1. Set a 90-minute timer
2. Solve 3 problems without any assistance
3. Review your solutions with Claude Code afterward
4. Identify patterns you struggled with
5. Re-study those patterns

## Tips for Success

1. **Read problems carefully** - Edge cases are often in the description
2. **Start with brute force** - Get something working first
3. **Optimize after correctness** - Don't premature optimize
4. **Test edge cases** - Empty arrays, single elements, duplicates
5. **Watch time complexity** - N ‚â§ 100,000 typically requires O(n) or O(n log n)
